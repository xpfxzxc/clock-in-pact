# AGENTS.md

## 项目概述

**打卡契约 / ClockInPact** - 社交监督式打卡与PK系统，一个轻量级、可自定义、具有社交监督与惩罚/奖励机制的打卡平台，帮助小团体（2-6人）坚持完成长期目标。

核心角色：
- **挑战者**：设定并执行目标，需要每日打卡
- **监督者**：负责审核证据、确认打卡有效性
